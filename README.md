# CNRoutingAssignment
 This repository contains the configurations and implementation of basic and advanced routing concepts using Cisco Packet Tracer.   

## 📌 Overview
The purpose of this assignment is to demonstrate practical knowledge of:
- Basic router configuration
- Static routing
- Dynamic routing protocols (RIP and EIGRP)

## ⚙️ Tasks Covered

### ✅ Task 1: Basic Router Configuration
- Configured router hostname
- Set console password and login authentication

### ✅ Task 2: Static Routing
- Designed a network with three routers and three LANs
- Configured IP addressing
- Implemented static routing between networks
- Verified connectivity using ping

### ✅ Task 3: Dynamic Routing
#### 🔹 RIP Configuration
- Configured RIP version 2 on all routers
- Disabled auto-summary
- Verified routing tables and connectivity

#### 🔹 EIGRP Configuration
- Removed RIP configuration
- Configured EIGRP (AS 100)
- Verified routing and communication

## 🌐 Network Design
- LAN Networks:
  - 192.168.1.0/24
  - 192.168.2.0/24
  - 192.168.3.0/24
- Point-to-Point Links (/30 subnets):
  - 10.0.12.0/30
  - 10.0.13.0/30
  - 10.0.23.0/30

## 📁 Repository Contents
- Packet Tracer files (.pkt)
- Configuration files (.txt) for each router and task
- Screenshots of routing tables and ping results
- Assignment report (PDF)
